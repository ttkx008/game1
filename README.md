# 🎮 3D Voxel Game

基于 Three.js 的 3D 体素游戏。

## 在线体验

访问 [GitHub Pages](https://ttkx008.github.io/game1/) 即可在线游玩。

## 文件说明

- `index.html` - 游戏入口页面
- `game_final.html` - 完整游戏版本
- `game.html` - 游戏主文件
- `three.min.js` - Three.js 库（压缩版）
- `three_local.js` - Three.js 本地版本

## 技术栈

- Three.js - 3D 渲染引擎
- HTML5 Canvas
